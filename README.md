# training

Kleines Next.js-Projekt zur Trainingsübung.

Schnellstart

1. Node.js installieren
2. Abhängigkeiten installieren:

```bash
npm install
```

3. Dev-Server starten:

```bash
npm run dev
```

Weitere Hinweise

- Projektstruktur: `app/`, `components/`
- Ignorierte Dateien: siehe `.gitignore`

Lizenz: siehe Repo auf GitHub
